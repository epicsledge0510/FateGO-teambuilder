Project Proposal

This website seeks to give users the ability to build their own teams based on a web database that keeps up to date information about the game.

Project requirements

Resources utilized

w3schools.com

Atlas Academy API developer discord server

API

Atlas Academy API

This API seeks to allow developers to create apps that use up to date information for the Mobile RPG Fate/Grand Order. The API is community made and uses a system that scrapes the APK used to install updates in order to find information about new characters and other aspects of the game in order to stay up to date.

Noteworthy

API loading

Using a fucntion, this program uses fetch in order to load information from a web based API and convert it into a JSON. This JSON is then used to parse information when searched

Team Building

This program allows the user to add characters from the game into a team, as well as sharing it tothe cloud if desired.

Firebase Functionality

Firebase cloud functionality is used to load user created teams. This can be used by users to submit their teams, while other users can use them as a basis in their own team building process
